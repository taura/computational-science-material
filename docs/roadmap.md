# 学習ロードマップ — 技法と応用のつながり, 面白い到達目標

このコースは, 前半 (`02_parallel`〜`13_ilp`) で **高性能計算の技法**を学び, 後半 (`14_montecarlo`〜`19_pinn`) でそれらを使って **意味のある計算**に挑戦する。
このページは, 学んだ技法が応用問題でどうつながり, どんな**面白いゴール**に届くかを示す案内図である。

## 2. 技法から応用への流れ (5本の柱)

### 柱1: 行列積 → AI
```
03 行列ベクトル積 ┐
05 行列積        ┼─→ D1 MNIST推論 ─→ D3 回帰の学習 ─→ D4 MLP学習 ─→ F1 PINN
13 SIMDで行列積  ┘    (推論=行列積+活性化)  (勾配降下)    (forward+backprop)  (NNでPDE)
```
- D1 は「**自分が並列化した行列積が, そのまま AI の推論になる**」着地点。
- D3 → D4 で「学習」へ, F1 で「PDE を NN で解く」へと発展する。

### 柱2: stencil (近傍平均) → 偏微分方程式
```
03 collapse / 06 reduction ─→ B1 2D熱伝導(定常) ─→ B2 波動 ─→ B3 拡散
```
- B1 が stencil の主役 (collapse(2) + 残差の reduction)。B2/B3 は同じ仕組みを時間発展に拡張する。

### 柱3: 反復解法 (行列ベクトル積 + 内積) → 線形代数・固有値
```
03 行列ベクトル積 / 06 内積 ─→ G1 CG法 (Ax=b を解く)
                            └→ G2 べき乗法 ─┬→ ① 固有振動モード (定在波の形)
                                            └→ ② すごろくの定常分布 (一番止まるマス)
```
- G1 と G2 は**同じ反復構造** (行列ベクトル積 → 内積 → ベクトル更新 → 収束判定)。
- **B1 ↔ G1 ↔ G2① は同じ 2D ラプラシアン行列**: B1 で温度場を求め, G1 で同じ方程式を解き, G2① で同じ行列の固有振動モード (定在波) を出す。「**同じ行列を, 解く / 満たす / 振動させる**」。

### 柱4: モンテカルロ / 確率
```
06 モンテカルロπ ─→ A1 ガチャ(期待値・ばらつき) ─→ A4 パーコレーション(相転移)
                                     (04 schedule の動的負荷分散が効く)
```

### 柱5: O(N²) の相互作用 → N体・群れ
```
03 二重ループ / 06 reduction / 11-13 SIMD ─→ C1 N体(重力) ─→ C2 Boids(群れの創発)
```

## 3. おすすめの取り組み順 (前提を満たす, 易→難)

**A1 → A4 → B1 →(B2, B3)→ G1 → G2 → C1 → C2 → D1 → D3 → D4 → F1**

ざっくりの前提:
- A1: reduction / A4: reduction + schedule
- B1: collapse + reduction → B2, B3 はその拡張
- G1: 行列ベクトル積 + 内積 (B1 と同じ行列) / G2: G1 と同じ反復
- C1: 二重ループ + reduction → C2 はその応用
- D1: 行列積 → D3 (学習の入口) → D4 (学習の集大成) → F1 (最上級, D4 と B系の理解が前提)

---

## 4. 面白い到達目標

1. **行列積が AI になる**: 並列化した行列積 → D1 で手書き数字を認識 → D4 で自分の NN を学習させ正解率が上がる。
2. **同じラプラシアンを, 解く・満たす・振動させる**: B1 (差分法) → G1 (CG) → G2① (固有振動モード) → F1 (PINN)。1つの行列を軸に, 差分・反復解法・固有値・機械学習の4つの流儀を体験。
3. **相転移を目撃する**: A4 で確率 p を動かすと, 臨界点で浸透確率が急に立ち上がる。
4. **べき乗法で形と順位を出す**: G2① 定在波の形 (1Dは sin と一致, 2Dは太鼓の模様) / G2② すごろくで一番止まりやすいマス。
5. **宇宙と群れを動かす**: C1 (重力N体) / C2 (群れ) を SIMD・GPU で加速し, O(N²) の重い計算を実用速度に。

## 5. 「技法 × 応用」被覆マップ (どの応用でどの技法が主役か)

| 技法 | 主に使う応用問題 |
|---|---|
| parallel for / collapse | B1, B2, B3, C1, 行列系 |
| schedule (動的負荷分散) | A4 |
| reduction | A1, B1 (残差), C1, D3, G1, G2 |
| GPU (target/teams/map) | B1, C1, D1, G1 (発展課題として) |
| SIMD / ILP / ベクトル型 | C1, D1, G1 の高速化 |
| 反復 (収束まで回す) | B1, G1, G2, D3, D4, F1 |

各応用問題には「発展目標 (マルチコア → GPU → SIMD/ILP)」と「性能を比べる」セルがある。どこまで速くできるか挑戦してみよう。
