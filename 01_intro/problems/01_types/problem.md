# 練習問題: 数値型と割り算

## 目標

整数と浮動小数点の違い, とくに **整数同士の割り算は小数部が切り捨てられる** ことを体験する.

## 課題

`types.cpp` (または `types.f90`) は, 摂氏 `c` を華氏 `f` に変換する.
変換式は

```
f = c × 9/5 + 32
```

である. `TODO` の箇所にこの式を書け.

注意: `9/5` のように **整数同士で割ると 1 になってしまう** (本当は 1.8). 正しく計算するには, 少なくとも一方を浮動小数点で書く (C++ なら `9.0/5.0`, Fortran なら `9.0d0/5.0d0`).

わざと `c * 9 / 5 + 32` と整数で書いたらどうなるかも試して, 結果の違いを確認せよ.

## コンパイルと実行

```
# C++
nvc++ -fast types.cpp -o types.exe

# Fortran
nvfortran -fast types.f90 -o types.exe
```

```
./types.exe
```

## 期待される結果

```
36.5 C = 97.7 F
```

`c` の値を変えて, 正しく変換されることを確かめよ.
